# Klasifikasi Tingkat Obesitas (Machine Learning)

## Deskripsi
Proyek ini bertujuan mengklasifikasikan tingkat obesitas berdasarkan pola hidup dan aktivitas fisik menggunakan Machine Learning.

## Dataset
- 2158 data
- 16 fitur
- Target: Nobeyesdad (7 kategori obesitas)

## Preprocessing
- Missing value (mean/modus)
- Encoding
- Outlier (IQR)
- Scaling (StandardScaler)
- SMOTE (imbalance)

## Model
- Random Forest
- XGBoost
- Hist Gradient Boosting

## Hasil
- Random Forest: 87%
- XGBoost: 88%
- Hist Gradient Boosting: **92% (terbaik)**

## Kesimpulan
Model Hist Gradient Boosting memberikan performa terbaik dalam klasifikasi tingkat obesitas.
